# Pluzman - Telegram Topluluğu

Bu proje Pluzman'ın Telegram grup, profil ve bot hizmetlerini tanıtan modern bir GitHub Pages web sitesidir.

## 🚀 Özellikler

- **Modern Tasarım**: Koyu tema ve Telegram renklerini kullanan modern tasarım
- **Responsive**: Tüm cihazlarda mükemmel görünüm
- **Telegram Entegrasyonu**: Direkt Telegram bağlantıları
- **Smooth Animasyonlar**: Hover efektleri ve geçişler

## 📱 Telegram Hizmetleri

### Telegram Grubu
- **Bağlantı**: [https://telegram.me/pluzmna/](https://telegram.me/pluzmna/)
- **Açıklama**: Aktif topluluğumuzla etkileşime geçin, sorularınızı sorun ve deneyimlerinizi paylaşın.

### Kişisel Profil
- **Bağlantı**: [https://telegram.me/pluzman](https://telegram.me/pluzman)
- **Açıklama**: Doğrudan benimle iletişime geçin ve sorularınızı özel olarak sorun.

### Tavsiye Botu
- **Bağlantı**: [https://telegram.me/tavsiyeebot](https://telegram.me/tavsiyeebot)
- **Açıklama**: Gelişmiş botumuz size çeşitli tavsiyeler ve bilgiler sunmak için tasarlandı.
