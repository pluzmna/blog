Stream4Me

Un fork italiano di Alfa

Stream4Me è un add-on per Kodi per cercare e guardare contenuti sul web.

I proprietari di questo add-on non ospitano né distribuiscono nessuno dei contenuti visualizzabili tramite questo add-on né hanno alcuna affiliazione con i fornitori di contenuti.

Come installare Stream4Me (via installer)

  • Aggiungi alle sorgenti file di Kodi il seguente URL: https://stream4me.github.io/repo/ con il nome "S4Me repo"
  • Apri Kodi e vai in Add-ons -> installa da file zip -> S4Me repo
  • Seleziona S4Me-installer.zip e installalo
  • Ti uscirà la schermata di configurazione iniziale, se così non fosse vai in Add-ons e apri Stream4Me
  • Ora puoi goderti l'add-on!

Metodo diretto (via zip)

  • Scarica il file zip: cliccando qui
  • Apri Kodi e vai in Add-ons -> installa da file zip
  • Seleziona il file appena scaricato (S4Me-installer) e installalo
  • Ti uscirà la schermata di configurazione iniziale, se così non fosse vai in Add-ons e apri Stream4Me
  • Ora puoi goderti l'add-on!
